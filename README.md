# Curry/Verse

**Live Demo:** [https://snekoyama.github.io/curryverse2/](https://snekoyama.github.io/curryverse2/)

## Concept
本作「Curry/Verse」は、論理とユーモアが交錯する1,063に及ぶ独自の「カレー回文」コーパス（言語集積）を、空間的な構造へと昇華させたインスタレーション作品のデジタルアーカイブです。

画面上を浮遊するテキスト群は、鑑賞者のタップ（あるいはマウスカーソル）によって引力（Attractor）を生み出します。静的なテキストデータを、常に変容し続ける動的な視覚体験として再構築しています。

## Technologies
外部ライブラリ（p5.js等）に依存せず、あらゆる環境で軽量かつ独立して動作する設計思想で構築しています。

* **HTML5 Canvas**: パーティクルの描画および軌跡の蓄積
* **Vanilla JavaScript**: アニメーションループおよび状態管理
* **Perlin Noise**: 有機的な動きを生成するノイズアルゴリズム（独自実装）
